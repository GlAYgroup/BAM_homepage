# BAM — Book AI Mentor

AI を活用した新しい読書体験プラットフォーム「BAM（Book AI Mentor）」のホームページです。

## 概要

BAM は、好きな著者と本屋を回ってカフェで話せるような、まったく新しい読書体験を AI で実現するサービスです。

### BtoC — BAM ToC（AI リーダー）
書籍 EC プラットフォームで購入した電子書籍を、AI とリアルタイムで対話しながら読める電子書籍リーダー。複数の本を同時に入力して著者視点を比較・討論できます。

### BtoB — BAM ToB（AI 書店）
出版社が書籍を登録し、AI による試読チャットを提供できる EC 基盤。購入前に AI と本の内容について対話し、購入判断に活用できます。
